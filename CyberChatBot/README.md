📋 Project Description
A console based cybersecurity awareness chatbot desigined for South African citizens to learn abouth online safety practices.The bot provides interactive convesations about various cybersecurity topics including password safety, phishing awareness/prevention,malware protection,and safe browsing habits.


💻 Features Used
voice greeting: Playes a welcome message on startup using System.Media.SoundPlayer to enhance user experience.
ASCII Art Logo: Displays a custom CyberChatBot logo using ASCII art to create a visually appealing interface.
Personalized Chat: Asks for user's name and personalizes all responses to create a more engaging and interactive experience.
Cybersecurity Topics: Responds to questions about various cybersecurity topics such as password safety, phishing awareness/prevention, malware protection, and safe browsing habits.
Inputs Validation: Handles empty inputs, invalid entries, and unrecognized commands gracefully by providing appropriate feedback to the user.
Colorful UI: Uses colored text,borders, emojis, and structured formatting to make the chatbot visually appealing and engaging for users.
Modular Code: Seprated into multiple classes (Program, CyberChatBot, ResponseManager, UIManager, AudioManager) and methods to improve readability, maintainability, and scalability of the codebase.
